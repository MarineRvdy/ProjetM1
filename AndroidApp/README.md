# Clonage d'un dépot

* git clone <url_depot>

* Windows : exécuter ./download_tlflite_libs.bat contenu dans /app/src/main/cpp/tflite/

* Linux : exécuter ./download_tlflite_libs.sh contenu dans /app/src/main/cpp/tflite/




# Terminal : 

* git add <file>

* git commit -m "message"

* git push
